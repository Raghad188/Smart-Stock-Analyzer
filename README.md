# محلل الأسهم الذكي | Smart Stock Analyzer

واجهة هاكاثون احترافية لمنصة عربية مدعومة بالذكاء الاصطناعي تحلل أخبار السوق السعودي، وتوضح الأثر المتوقع على حركة الأسهم مع درجة ثقة وتفسير مبسط.

> هذا المشروع تطبيق عرض تجريبي. جميع الأسعار والأخبار والتحليلات بيانات محاكاة، ولا تمثل توصيات استثمارية.

## المزايا

- لوحة سوق فورية مع مؤشر تاسي، القطاعات، الأسهم النشطة والرابحين والخاسرين
- صفحات تفصيلية لأسهم أرامكو، الراجحي، سابك، إس تي سي، الأهلي وموبايلي
- تحليل أخبار ذكي مع نوع التأثير ودرجة الثقة وأسباب التوقع
- رسوم Area وBar تفاعلية ومتحركة
- سجل تحليلات مع البحث والتصفية والترتيب
- قائمة مفضلة محفوظة محلياً عبر Local Storage
- مركز إشعارات ومحاكاة للتنبيهات والتوصيات
- صفحة تعريفية متكاملة للمشروع والفريق
- Landing Page مخصصة لعرض الفكرة في الهاكاثون
- تصميم RTL متجاوب للحاسب واللوحي والجوال
- حركات انتقال وتفاعل مبنية باستخدام Framer Motion

## التقنيات

- Next.js 16 + App Router
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React
- Mock JSON + Local Storage

## التشغيل محلياً

يتطلب Node.js 20.9 أو أحدث.

```bash
npm install
npm run dev
```

ثم افتح [http://localhost:3000](http://localhost:3000). تفتح لوحة التحكم مباشرة من دون تسجيل دخول.

## أوامر المشروع

```bash
npm run dev     # تشغيل بيئة التطوير
npm run lint    # فحص جودة الكود
npm run build   # إنشاء نسخة الإنتاج
npm run start   # تشغيل نسخة الإنتاج
```

## هيكل المشروع

```text
app/          صفحات App Router والتنسيقات العامة
components/   مكونات الواجهة والرسوم التفاعلية
hooks/        المفضلة والتنبيهات المحلية
lib/          الأنواع، البيانات المجمعة، والأدوات
data/         بيانات JSON التجريبية
public/       الأصول العامة والشعار
assets/       أصول التصميم المصدرية
styles/       مساحة جاهزة للتوسعات النمطية
```

## الصفحات

| المسار | الوظيفة |
| --- | --- |
| `/` | لوحة السوق الرئيسية |
| `/welcome` | صفحة العرض والهبوط |
| `/stocks/[symbol]` | تفاصيل السهم |
| `/analysis` | تحليل الأخبار |
| `/history` | سجل التحليلات |
| `/favorites` | الأسهم المفضلة |
| `/notifications` | مركز الإشعارات |
| `/about` | قصة المشروع والفريق |

## النشر على Vercel

1. ارفع المشروع إلى مستودع Git.
2. استورده في Vercel.
3. اترك إعدادات Next.js الافتراضية.
4. لا توجد متغيرات بيئة أو مفاتيح API مطلوبة.

يمكن أيضاً استخدام:

```bash
npx vercel
```

## ملاحظات العرض

- صفحة البداية هي لوحة التحكم لتقليل زمن وصول لجنة التحكيم إلى القيمة الأساسية.
- صفحة `/welcome` مناسبة لبدء العرض التقديمي وسرد قصة المنتج.
- كل الأزرار الأساسية تؤدي إلى صفحة، تحديث مرئي، أو تفاعل محاكاة واضح.
- المفضلة تبقى محفوظة في المتصفح بعد إعادة التحميل.
